# 🔭 OSINT Arsenal

A curated collection of OSINT tools and resources, built as a fast static web app.

**Live:** https://alexanderweronka.github.io/Osint-Arsenal
---

## Features

- 🛠️ **Tools** — searchable, filterable database of OSINT instruments
- 🎲 **Random Tool** — discover lesser-known tools with descriptions
- 📚 **Knowledge Base** — guides on OSINT methodology
- 🎯 **Practice** — platforms and CTF challenges
- 📖 **Resources** — books, channels, communities
- 🌐 **RU / EN** — bilingual interface

## Stack

Pure HTML + CSS + JS. No build step, no dependencies. Works offline after first load.

## Deploy

Hosted on **GitHub Pages** — just push to `main`, it goes live automatically.
